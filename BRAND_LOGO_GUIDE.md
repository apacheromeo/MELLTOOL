## Important: Copyright & Trademarks

### ‚ö†Ô∏è Legal Considerations

**Why we can't automatically download logos:**
1. Brand logos are **copyrighted** and **trademarked**
2. Unauthorized use can violate intellectual property laws
3. Each brand has specific usage guidelines

**Legal ways to use brand logos:**
1. **Official Press Kits**: Many brands provide logos for authorized retailers
2. **Retailer Portals**: If you're an authorized dealer, use their assets
3. **Fair Use**: As a retailer selling legitimate products, you can display brand logos for identification purposes
4. **Permission**: Contact brands directly for logo usage rights

### Safe Usage Guidelines

‚úÖ **Acceptable**:
- Using official logos from brand press kits
- Displaying logos for products you legitimately sell
- Identifying brands accurately in your inventory

‚ùå **Avoid**:
- Modifying or altering brand logos
- Using logos to imply false endorsement
- Downloading from random websites (may be low quality or modified)

## Step-by-Step: Adding Logos to Brands

### Option A: Upload Image Logo (Recommended for Professional Look)

**Step 1: Prepare Logo File**
- Format: PNG (with transparent background) or JPG
- Size: Under 2MB
- Recommended dimensions: 500x500px or smaller
- Square aspect ratio works best

**Step 2: Navigate to Brands**
1. Login to MELLTOOL
2. Click **"Inventory"** in sidebar
3. Click **"Brands"**

**Step 3: Edit Brand**
1. Find the brand you want to add a logo to
2. Click the **"Edit"** button (pencil icon)

**Step 4: Upload Logo**
1. In the modal, find **"Logo/Icon"** section
2. Click **"Upload Image"** button
3. Click **"Upload Logo"** button
4. Select your logo file
5. Preview will appear - verify it looks good
6. Click **"Update Brand"**

**Step 5: Verify**
- Logo should now appear in the brand list
- Test by going to POS ‚Üí Brand Selection
- Logo should display in the brand grid

### Option B: Use Emoji Logo (Quick Method)

**When to use**:
- Generic or unknown brands
- Placeholder until you get the real logo
- Small or local brands without official logos

**Steps**:
1. Edit the brand
2. Keep **"Use Emoji"** selected
3. Choose an appropriate emoji from the grid:
   - üè∑Ô∏è Generic brand
   - üì± Electronics
   - ‚ö° Electrical/Tech
   - üí° Lighting
   - üîß Tools/Hardware
   - üì∫ TV/Monitor brands
   - üíº Professional/Business brands
4. Click **"Update Brand"**

## Adding Logos for Your Existing Brands

Here's a practical workflow for adding logos to all your brands:

### Workflow

**Day 1-2: Identify Brands**
```bash
# Get list of all your brands
1. Go to Inventory ‚Üí Brands
2. Export or screenshot the list
3. Count how many need logos
```

**Day 3-5: Gather Logo Files**
1. Group brands by source (e.g., all electronics together)

1. Go through each brand alphabetically
2. Edit ‚Üí Upload Image ‚Üí Select file ‚Üí Save
3. Verify logo appears correctly
4. Move to next brand

**Time estimate**: ~2-5 minutes per brand

### Batch Processing Tips

**For 10-20 brands:**
- Set aside 1-2 hours
- Prepare all logos first
- Then upload all at once

**For 50+ brands:**
- Do in batches of 10
- Prioritize top-selling brands first
- Use emojis for less important brands temporarily

## Troubleshooting

### Logo Doesn't Display

**Problem**: Uploaded logo but it doesn't show

**Solutions**:
1. **Check file size**: Must be under 2MB
   - If too large, use image compression tool: [TinyPNG](https://tinypng.com)
2. **Check file format**: Must be JPG or PNG
   - Convert if needed: [CloudConvert](https://cloudconvert.com)
3. **Clear browser cache**: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
4. **Re-upload**: Delete and upload again

### Logo Looks Pixelated

**Problem**: Logo appears blurry or low quality

**Solutions**:
1. Find higher resolution version (minimum 300x300px)
2. Download from official source
3. Avoid screenshotting logos - download the actual file

### Logo Has Wrong Background

**Problem**: Logo has white/colored background instead of transparent

**Solutions**:
1. Look for PNG version with transparency
2. Use online background remover: [Remove.bg](https://remove.bg)
3. Ask brand representative for PNG with transparent background

### Logo Too Large

**Problem**: File size exceeds 2MB

**Solutions**:
1. Compress image: [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app)
2. Resize to smaller dimensions (500x500px is sufficient)
3. Convert to PNG if it's JPG (PNG with transparency is often smaller)

### Can't Find Official Logo

**Solutions**:
1. Use emoji placeholder temporarily
2. Contact brand sales representative
3. Check if you're an authorized dealer - access dealer portal
4. Use Wikipedia/Wikimedia Commons version

## Best Practices

### Logo Quality Standards

‚úÖ **Recommended**:
- **Format**: PNG with transparent background
- **Dimensions**: 500x500px (square)
- **File size**: Under 500KB (ideally under 200KB)
- **Quality**: Clear, crisp, official version

### Consistency

**For Professional Look**:
1. All logos should be official versions
2. Consistent background (transparent preferred)
3. Proper aspect ratio (square works best)
4. Clear and readable at small sizes

### Naming Convention (for your downloaded files)

**Good naming**:
```
samsung-logo.png
lg-logo.png
dyson-logo.png
```

**Avoid**:
```
Image001.png
logo.png
download (1).png
```

### Backup Your Logo Files

**Important**: Keep copies of all logo files
1. Create folder: `brand-logos-backup/`
2. Save all downloaded logos there
3. Include source/date in a text file
4. This helps if you need to re-upload later
